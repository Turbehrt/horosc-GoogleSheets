Project authors
==========

The application Horosc for Google Sheets is based on John D. North's historical HOROSC program described in his book: John D. North, _Horoscopes and History_, London: Warburg Institute, 1986, appendix 4, pp. 197-218.

The code has been entirely rewritten in Javascript for Google Sheets (Google Apps Script) by **François J. Tur** in 2021.

**Alexandre Tur** provided scientific advice on the historical astronomy concepts used, conducted tests on historical material, and produced documentation.

Auteurs du projet
==========

L'application Horosc pour Google Sheets est basée sur le programme historique HOROSC de John D. North décrit dans son ouvrage : John D. North, _Horoscopes and History_, Londres : Warburg Institute, 1986, annexe 4, pp. 197-218.

Le code a été entièrement réécrit en Javascript pour Google Sheets (Google Apps Script) par **François J. Tur** en 2021.

**Alexandre Tur** a fourni des conseils scientifiques sur les concepts d'astronomie historique utilisés, a effectué des tests à partir de sources historiques et a produit la documentation.
